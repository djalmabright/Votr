## Installation

```bash
https://github.com/djalmabright/Votr.git
```

Install requirements

```bash
pip install -r requirements.txt
```

**set the FLASK_APP and FLASK_DEBUG variables**

```bash
export FLASK_APP=votr.py
export FLASK_DEBUG=1
```

Finally run it with:

```bash
flask run
```
